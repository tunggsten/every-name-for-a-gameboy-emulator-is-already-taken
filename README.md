# A Game Boy emulator I'm making to learn Rust programming

This is not going to be an enormously impressive project; however, making an emulator and learning Rust have both been on my list for ages now, so I decided I'd get on with both.
Hopefully, this will be an introduction to proper assembly programming (albeit for a processor architecture that literally only one thing uses), interpreter structures, WGPU and possibly LLVM.
If I don't make it that far though, this has still been a very fun project to work on so far!
